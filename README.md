# EnergyM 💪

Hi 👋, welcome to **EnergyM** – a fitness & wellness web platform! 

Hi 👋, I'm Rohit Gupta
🎓 B.Tech Student | 💻 Frontend Developer
🌐 Passionate about building clean & responsive web interfaces

This project showcases professional fitness programs, trainers, and interactive program pages built with **HTML, Tailwind CSS, and JavaScript**.  

---

## 🚀 Features

- Modern, responsive design with Tailwind CSS
- 
- Interactive program pages (Cardio, CrossFit, Boxing, Yoga, etc.)
- 
- Trainers section with hover effects and social links
- 
- Mobile-friendly navigation with hamburger menu
- 
- Smooth scroll animations & reveal effects
- 
- Dashboard/Login: Secure user area to track progress, manage profiles, view personalized programs, interact with trainers, and monitor fitness goals.
